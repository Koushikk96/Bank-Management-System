# Bank Management System

## Overview

A console-based C++ application that simulates core banking operations using Object-Oriented Programming (OOP) and File Handling.

## Features

- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Persistent Data Storage
- Menu-Driven Interface

## Technologies Used

- C++
- OOP Concepts
- File Handling

## How to Compile

```bash
g++ main.cpp -o bank
```

## How to Run

```bash
./bank
```
